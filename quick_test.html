  • 📌 Чанкинг диалога: текущий chunk_size=3, overlap=1 при коротком диалоге создаёт дублирующиеся чанки. Рассмотри использование только последнего сообщения для эмбеддинга.

  • 📌 Для чувашскоязычных вопросов рассмотри Hypothetical Document Embeddings (HyDE): сначала генерируй гипотетический ответ, затем ищи по нему в ChromaDB.
